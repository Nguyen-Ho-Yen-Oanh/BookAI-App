# 📚 BookAI v2.0

## Tính năng
- 🤖 AI gợi ý sách theo sở thích (Google Gemini - miễn phí)
- ⚙️ Admin quản lý kho sách (thêm, sửa, xóa)
- 📖 Đọc sách online (Google Books, Gutenberg, Tiki)
- 🔍 AI phân tích & tóm tắt nội dung sách
- 💾 Dữ liệu lưu tự động vào trình duyệt

## Cách chạy

### Bước 1 - Lấy API key miễn phí
Vào https://aistudio.google.com/apikey → Đăng nhập Google → Create API key → Copy key

### Bước 2 - Điền API key
Mở file `.env` → thay `PASTE_KEY_HERE` bằng key của bạn → Lưu

### Bước 3 - Chạy app
Double-click `CHAY_APP.bat` (Windows)
Hoặc mở Terminal: `npm install` rồi `npm run dev`

Mở trình duyệt: http://localhost:3000
